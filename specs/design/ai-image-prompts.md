## Dicas de Uso dos Prompts

### DALL-E 3 (OpenAI)
- Use descricoes claras e especificas
- Mencione cores em formato hexadecimal
- Especifique tamanhos desejados
- Adicione "no text" para evitar texto indesejado

### Midjourney
- Use parametros como --ar (aspect ratio), --s (stylize), --tile (para patterns)
- --style raw para resultados mais precisos
- --s 50-100 para designs mais controlados
- --s 200+ para mais criatividade artistica

### Leonardo.AI
- Boa opcao para icons e UI elements
- Use modelo "DreamShaper" para ilustracoes
- Use modelo "Absolute Reality" para texturas

### Canva AI / Adobe Firefly
- Boas alternativas para assets comerciais
- Licenciamento mais claro para uso comercial

## Workflow Recomendado

1. **Gerar com IA**: Use os prompts acima para gerar variacoes
2. **Selecionar**: Escolha as melhores opcoes
3. **Refinar**: Use editores como Figma ou Photoshop para ajustes
4. **Otimizar**: Comprima para web (TinyPNG, SVGO)
5. **Implementar**: Integre nos arquivos CSS/HTML
